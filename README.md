## About the Tool

BTXpred is a bioinformatics-based prediction system designed to identify bacterial toxins directly from amino acid sequences. The method employs multiple computational approaches including Support Vector Machines (SVM), Hidden Markov Models (HMM), and PSI-BLAST for accurate toxin classification.

The system was trained and validated on a non-redundant dataset containing both exotoxins and endotoxins.

---

## Key Features

### Bacterial Toxin Prediction

* Predicts whether a protein sequence is a bacterial toxin
* Sequence-based computational analysis

### Toxin Classification

* Differentiates:
  * Exotoxins
  * Endotoxins

### Exotoxin Family Identification

Classifies exotoxins into categories such as:

* Neurotoxins
* Adenylate cyclase activating toxins
* Guanylate cyclase activating toxins

### Machine Learning Models

* Support Vector Machine (SVM)
* Hidden Markov Model (HMM)
* PSI-BLAST similarity search

BTXpred utilizes:

* Amino acid composition
* Dipeptide composition
* Sequence similarity analysis
* Machine learning classification models

The developed models achieved high prediction accuracies for toxin identification and classification.

---

## Dataset Information

The study used:

* 150 bacterial toxin sequences
  * 77 exotoxins
  * 73 endotoxins

The dataset was non-redundant and experimentally validated.
